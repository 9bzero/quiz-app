# Quiz App

Interactive quiz game with countdown timer, instant feedback, and final score breakdown.

## Features

- Multiple categories and difficulty levels
- 30-second countdown per question
- Instant correct/incorrect feedback with answer reveal
- Score tracker and accuracy percentage
- Results screen with per-question review
- Shuffle questions and answers on each attempt
- Save high scores to localStorage
