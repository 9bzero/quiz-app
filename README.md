# Quiz App

An interactive multiple-choice quiz game with categories, timer, and scoring — built with React and TypeScript.

## Features
- Three quiz categories: JavaScript, CS Fundamentals, Web Dev
- 20-second countdown timer per question (color changes as time runs out)
- Instant feedback with correct/incorrect highlighting
- Explanation shown after each answer
- Score summary with question-by-question review
- Questions randomized each round

## Tech Stack
React 18 · TypeScript · Tailwind CSS v4 · Vite

## Getting Started
```bash
npm install
npm run dev
```
